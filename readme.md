Simple Framebuffer Graphics Library
===================================
简易帧缓冲图形库
----------------

###支持系统
>  
    Linux
        
###需求
>       
    内核有framerbuffer驱动支持
    支持32位色彩模式
    root permission
        
###加载函数库
>       
    libfreetype
    libpng
    libm
    libpthread
